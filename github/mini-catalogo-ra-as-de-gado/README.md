

# 🐄 Catálogo de Raças de Gado Mais Produzidas no Brasil

Projeto educacional desenvolvido em HTML puro para apresentar as principais raças de gado criadas no Brasil, com informações detalhadas e vídeos explicativos sobre cada raça.

---

## 📋 Sobre o Projeto

Este catálogo foi criado como projeto de estudo de **HTML5 semântico**, apresentando 6 raças bovinas amplamente criadas no Brasil: Nelore, Angus, Brahman, Senepol, Sindi e Guzerá.

O projeto demonstra a aplicação prática de tags semânticas, estruturação de conteúdo e navegação entre páginas HTML.

---


## 🚀 Demonstração

O projeto consiste em:
- **Página principal:** Catálogo com todas as raças, imagens e descrições
- **Páginas individuais:** Uma página para cada raça com vídeo informativo do YouTube

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura e conteúdo
- **CSS3** - Estilização básica (inline styles)
- **YouTube Embed** - Vídeos informativos sobre cada raça

---

## 📌 Observações

⚠️ **Importante:** Este é um projeto educacional desenvolvido para fins de aprendizado de HTML5. As informações sobre as raças são baseadas em conhecimento geral sobre pecuária brasileira.
